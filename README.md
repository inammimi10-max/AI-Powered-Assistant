# AI Workplace Productivity Assistant

## Project Overview

AI Workplace Productivity Assistant is a modern web application designed to help professionals automate common workplace tasks using Artificial Intelligence. The application provides an intuitive dashboard where users can generate professional emails, summarize meeting notes, plan tasks, conduct research, and interact with an AI chatbot.

The goal of this project is to improve workplace productivity by reducing repetitive work and enabling users to complete tasks more efficiently through AI-powered assistance.

---

## Features

### 📧 Smart Email Generator
- Generate professional emails instantly.
- Select recipient, tone, and email length.
- Add custom instructions.
- Edit, copy, download, regenerate, and save generated emails.

### 📝 Meeting Notes Summarizer
- Convert lengthy meeting notes into concise summaries.
- Highlight key decisions and action items.
- Save time reviewing meeting discussions.

### ✅ AI Task Planner
- Create structured daily or weekly task plans.
- Prioritize tasks based on urgency.
- Improve productivity and time management.

### 🔍 AI Research Assistant
- Generate research summaries on various topics.
- Organize information into easy-to-read sections.
- Save valuable research time.

### 💬 AI Chatbot Interface
- Ask workplace-related questions.
- Receive AI-generated responses.
- Supports general productivity and work assistance.

### 🎨 Modern Dashboard
- Responsive layout for desktop and mobile devices.
- Sidebar navigation.
- Clean SaaS-inspired user interface.
- Dark mode friendly design.

### ⚠ Responsible AI Disclaimer
- Reminds users that AI-generated content should be reviewed before use.
- Encourages responsible and ethical use of AI.

## Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/ai-workplace-productivity-assistant.git
```

### 2. Navigate to the project folder

```bash
cd ai-workplace-productivity-assistant
```

### 3. Install dependencies

```bash
npm install
```

### 4. Configure environment variables

Create a `.env` file in the project root and add your OpenAI API key:

```env
OPENAI_API_KEY=your_api_key_here
```

### 5. Start the development server

```bash
npm run dev
```

or

```bash
npm start
```

### 6. Open the application

Visit:

```
http://localhost:3000
```

or

```
http://localhost:5173
```

depending on your development environment.
